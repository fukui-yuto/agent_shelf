# agent-shelf Example Knowledge

This is an example knowledge document for the agent-shelf framework.

## What is agent-shelf?

agent-shelf is a convention-over-configuration AI agent framework.
You create an agent by simply placing files in a directory structure:

- `knowledge/` - Documents for RAG (Retrieval Augmented Generation)
- `skills/` - Python scripts that the agent can execute as tools
- `prompts/` - System prompts and persona definitions

No configuration files are needed. The directory structure IS the configuration.
